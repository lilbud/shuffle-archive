# Site Data

This folder contains a number of JSON data files pertaining to the E Street Shuffle site. These are all pulled from the sites API.

Below is a list of and brief description of the files in this folder:

- `categories_20260114.json`: A list of site categories from January
- `categories.json`: A version of the above from April
- `comments_20260114.json`: All site comments from January
- `comments_april26.json`: Version of the above, but pulled in April
- `comments_complete.json`: A merged version of the above 2 files
- `tags.json`: A list of all tags present on the site
- `wayback_api.json`: A list of all JSON files present on the Wayback Machine. This was used to pull earlier versions of many posts.
- `wp_media_library.json`: A file containing nearly every file uploaded to the site.
